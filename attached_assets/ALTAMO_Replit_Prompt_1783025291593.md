# ПРОМТ ДЛЯ REPLIT AGENT — ИНТЕРНЕТ-МАГАЗИН ALTAMO

Скопируй всё ниже и вставь в Replit Agent одним сообщением.

---

Создай интернет-магазин премиальной посуды и предметов сервировки **ALTAMO** (люксовый сегмент, Москва). Стек: React + Vite + Tailwind CSS, роутинг через react-router-dom, состояние корзины/избранного через Context API + localStorage. Данные товаров — в JSON-файле (mock, 24–36 товаров), чтобы позже легко заменить на API.

## СТИЛЬ (строго соблюдать)

- Эстетика люксового бутика: минимализм, много воздуха, крупные отступы.
- Палитра: белый фон #FFFFFF, светло-серый фон секций #F2F2F0, тёмно-серый текст #1A1A1A, никаких ярких цветов.
- Заголовки: тонкий serif-шрифт (Cormorant Garamond из Google Fonts), UPPERCASE, увеличенный letter-spacing (0.2em–0.35em) — как «З А Г О Л О В О К».
- Основной текст и навигация: гротеск (Inter или Manrope), мелкий кегль, uppercase в меню, letter-spacing 0.1em.
- Логотип в шапке: текст «A L T A M O» с разрядкой, по центру.
- Кнопки: прямоугольные, тонкая рамка 1px, без скруглений, текст uppercase; hover — инверсия (тёмный фон, белый текст).
- Фото товаров: на светло-сером фоне, квадратные карточки. Пока используй серые плейсхолдеры.
- Анимации: деликатные (fade-in при скролле, плавный hover на карточках). Ничего кричащего.

## ШАПКА (на всех страницах)

1. Верхняя тёмная полоса: «Экспресс-доставка по Москве от 3-х часов».
2. Строка: слева телефон +7 (999) 999-99-99, по центру логотип ALTAMO, справа иконки: Поиск, Войти, Wishlist, Корзина (со счётчиком).
3. Меню: КАТАЛОГ · БРЕНДЫ · ПОДАРКИ · СЕРВИЗЫ · СЕРВИРОВКА · ДЛЯ БАРА · СТОЛОВЫЕ ПРИБОРЫ · ПРЕДМЕТЫ ИНТЕРЬЕРА · ЭКСКЛЮЗИВНЫЕ ИЗДЕЛИЯ.
4. У пунктов меню выпадающие подменю второго уровня:
   - Подарки: для неё, для него, на свадьбу
   - Сервизы: сервизы, тарелки, салатники, кружки и чашки, чайники и кофейники
   - Сервировка: сервировка осн. блюд, закусок, десертов, специи, соусы, прочее
   - Для бара: бокалы, фужеры, рюмки, стопки, стаканы, ёмкости, охлаждение
   - Столовые приборы: наборы, индивидуальные, для сервировки блюд, для стейка, сервировка десертов
   - Предметы интерьера: вазы и чаши, предметы, текстиль, подсвечники, фарфор
5. Шапка sticky. Мобильная версия — бургер-меню.

## СТРАНИЦЫ

### 1. Главная (/)
Секции сверху вниз:
1. Hero-слайдер (3 слайда): надзаголовок, крупный заголовок с разрядкой, краткое описание, кнопка. Точки-индикаторы.
2. «ВЫБОР ALTAMO» — карусель из 4+ карточек товара (бренд, название, цена «100 000 RUB»), стрелки, кнопка «В каталог».
3. Баннер 50/50 (текст слева, изображение справа): эксклюзивная коллекция / идеи подарков / сервизы, кнопка ведёт в каталог.
4. Секция категорий — горизонтальный ряд крупных плиток «КАТЕГОРИЯ», клик → каталог с фильтром.
5. Баннер «Подарочные карты» (изображение слева, текст справа).
6. «НОВОЕ ПОСТУПЛЕНИЕ» — карусель товаров + «В каталог».
7. Баннер «Ивенты / мастер-классы / лекции» → страница Мероприятия.
8. «БРЕНДЫ» — бегущая строка (marquee) логотипов/названий брендов, клик → каталог с фильтром по бренду.

### 2. Каталог / Категория (/catalog, /catalog/:category)
- Hero-полоса с названием категории, хлебные крошки «Главная / Каталог / Категория».
- Слева панель фильтров: Категории (ссылки), Бренды (поиск + чекбоксы), Тип товара, Материал, Цвет (Белый, Бежевый, Синий, Красный), Коллекция, Стоимость (от 1 500 до 1 500 000, два поля), кнопки «Применить» и «Очистить фильтры».
- Справа сверху «Сортировка» (по цене ↑↓, по новизне).
- Сетка товаров 3 колонки (мобайл — 2/1), карточка: фото, бренд, НАЗВАНИЕ ИЗДЕЛИЯ, цена; иконка избранного на hover.
- Кнопка «Загрузить ещё» (пагинация по 9).
- Фильтры реально работают по mock-данным.

### 3. Карточка товара (/product/:id)
- Хлебные крошки. Слева: большое фото + ряд миниатюр (4 шт., переключение).
- Справа: НАЗВАНИЕ ИЗДЕЛИЯ, цена, кнопка «ДОБАВИТЬ В КОРЗИНУ» (тёмная, широкая), под ней «Wishlist» и «Купить в 1 клик» (рамочные).
- Атрибуты списком: Бренд, Страна бренда, Страна производства, Коллекция, Материал, Тип.
- Табы: «Состав и описание» / «Уход за изделием» / «Доставка».
- Ниже карусели: «ПОХОЖИЕ ТОВАРЫ» и «РЕКОМЕНДАЦИИ ДЛЯ ВАС» (по 4 карточки + «В каталог»).

### 4. Бренды (/brands)
- Алфавитный указатель + поиск, список брендов по буквам, клик → каталог, отфильтрованный по бренду.

### 5. Мероприятия (/events, /events/:id)
- Заголовок «МЕРОПРИЯТИЯ», табы «Предстоящие / Архивные».
- Список: чередующиеся блоки 50/50 (текст ↔ изображение), надзаголовок, заголовок, описание, кнопка.
- Детальная: заголовок («ЦВЕТОЧНЫЕ КОМПОЗИЦИИ»), дата и время, два абзаца описания, кнопка записи → ссылка на WhatsApp (wa.me), галерея фото, блок контактов, кнопка «На главную».

### 6. Контакты (/contacts)
- «КОНТАКТЫ»: Телефон, Почта (altamo@list.org), Адрес пространства (г. Москва), «Связаться с нами»: WhatsApp, Telegram; «Социальные сети»: Pinterest, Telegram, Instagram*.
- Секция «ПРОСТРАНСТВО ALTAMO»: большое фото + 3 абзаца текста + галерея.

### 7. Оплата и доставка (/payment-delivery)
- Заголовок, табы «Оплата / Доставка». Тексты про оплату картами Visa/MasterCard/МИР, причины отказа платежа, оплату при получении. Для доставки — заглушка-текст.

### 8. Подарочные карты (/gift-cards)
- Баннер + выбор номинала карты, кнопка «В корзину».

### 9. Корзина (/cart), Избранное (/wishlist)
- Список позиций, изменение количества, удаление, итоговая сумма, кнопка «Перейти к оформлению» (форма-заглушка: контакты, адрес, способ получения).

### 10. Вход/Регистрация (/login)
- Вход по номеру телефона или почте, форма регистрации. Без реального бэкенда — просто UI + сохранение в localStorage.

## ФУТЕР (на всех страницах)
Светло-серый фон, логотип ALTAMO по центру, 4 колонки:
1. Навигация: Каталог, Бренды, Подарки, Сервизы, Сервировка, Для бара, Столовые приборы, Предметы интерьера, Эксклюзивные изделия.
2. Покупателям: О бренде, Подарочные карты, Мероприятия, Оплата, Доставка, Возврат, Юридические документы.
3. Контакты: Адреса магазинов, Телефон, Почта, Соцсети.
4. Подписка: «Узнавайте первыми об эксклюзивных поступлениях», поле Email + «Подписаться».
Внизу: «© 2025» слева, «Разработано CSG» справа.

## ДАННЫЕ
Создай products.json: id, название, бренд (Vista Alegre, Christofle, Baccarat, Hermès, Meissen, Ginori 1735), категория, подкатегория, цена (15 000 – 1 200 000 RUB), материал (хрусталь, фарфор, серебро, сталь), цвет, коллекция, страна, описание. Пример описания возьми в стиле: «Набор для водки (графин и 4 рюмки) Vista Alegre выполнен из высококачественного хрусталя и декорирован позолотой…»

## ТРЕБОВАНИЯ
- Полная адаптивность (desktop / tablet / mobile).
- Корзина и избранное работают и сохраняются в localStorage, счётчики в шапке обновляются.
- Поиск по каталогу (по названию и бренду) через иконку в шапке.
- Весь интерфейс на русском языке.
- Код структурируй по компонентам (Header, Footer, ProductCard, FilterPanel и т.д.), чтобы легко дорабатывать.
